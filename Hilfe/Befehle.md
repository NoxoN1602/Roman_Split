## /help (laut Orchestrator: „nur Liste aller vorgesehenen Kommandos“, Format: Phase, Kommando → Bedeutung)

**Allgemein**

- Allgemein, `/help` → Liste aller vorgesehenen Kommandos (Phase, Kommando → Bedeutung)
    
- Allgemein, `/missing_files` → Prüft KANON__, PLOT__, SCENES__, CHAR__, LOC__, REL__ inkl. Verweise; listet fehlende Dateien
    

**Phase 1 – Kanon**

- Phase 1, `/kanon_world` → Welt/Kanon erarbeiten
    
- Phase 1, `/kanon_character` → Charakter/Kanon erarbeiten
    
- Phase 1, `/kanon_location` → Ort/Kanon erarbeiten
    
- Phase 1, `/kanon_plot_status` → Plot-Status/Kanon erarbeiten
    

**Phase 2 – Plot**

- Phase 2, `/plot_overview` → Plot-Überblick
    
- Phase 2, `/plot_act` → Akt-/Dramaturgie-Entwicklung
    
- Phase 2, `/plot_decision` → Orchestrator-Entscheidung
    

**Phase 3 – Szenen**

- Phase 3, `/scene_idea` → Szenenideen
    
- Phase 3, `/scene_contract` → Szenenvertrag erstellen
    
- Phase 3, `/scene_write` → Szene schreiben (mit Kanonprüfung)
    

**Phase 4 – Review**

- Phase 4, `/review_consistency` → Konsistenzprüfung
    
- Phase 4, `/review_lektorat` → Lektorat (zukünftiger Agent)
    

**Erweiterung – Beziehungen**

- (Erweiterung), `/rel_create` → Neue Beziehung entwerfen
    
- (Erweiterung), `/rel_deepen` → Bestehende Beziehung vertiefen/analysieren
    
- (Erweiterung), `/rel_arc` → Beziehungsentwicklung über mehrere Szenen (Beziehungsbeats)
    
- (Erweiterung), `/rel_update_suggest` → Vorschläge für kanonische Beziehungsupdates
    

Wenn du jetzt ein Slash-Command gibst (z. B. `/scene_idea …`), kann ich als nächstes gemäß Orchestrator die nötigen Dateien per Obsidian-Linkauflösung nachladen und die Agenten-Kette starten.
